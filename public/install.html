Installation on Windows

Leiningen comprises basically two parts: a script and a standalone jar file. To install leiningen on Windows follow these steps:

  1. Download the leiningen script. This is a .bat file which you can call from the command prompt.

    Detailed instruction

    The default name of this script is lein. If you are working with multiple versions of leiningen you are free to rename this script for a specific version, e.g. lein2. On the third line of the script there is a version variable. This is set to the newest stable version by default. You can change it if you want to work with a specific version of leiningen for whatever reason.

  2. Place the script on your PATH.

    Detailed instruction

    One way to do this is to make a directory for the leiningen script to reside in, e.g. c:\Program Files\leiningen. Place the script in the directory.

    Then change the PATH environment variable by appending ;c:\Program Files\leiningen to it. More details on how to change the PATH variable per version of Windows can be found here.
  3. Download wget or curl and make it available on the PATH.

    Detailed instruction

    Leiningen needs an HTTP client that can download the leiningen standalone jar file. On *nix-systems such a client is already available. On Windows this isn't the case, so we need to install it. You can skip the installation of an HTTP client and manually download the jar file from github, but letting the script take care of that will make life a little easier in the long run. The script uses either wget or curl to do this. You can make a separate directory for this software and place it on the PATH like we did with the script in step 2, but since wget.exe is a single standalone file, I tend to just place it in C:\Program Files\leiningen, since that directory is on the PATH already.

  4. Open a command prompt and run lein self-install.

    Detailed instructions

    Open a command prompt by choosing from the Start menu > Run and then enter cmd.exe. A command prompt will now show up. Enter the command lein self-install. The script will now download the standalone jar file into a directory %USERPROFILE%/.lein/self-installs. The environment variable %USERPROFILE% is an is set to your home directory. If you chose to manually download a standalone jar file from github, you need to move it %USERPROFILE%/.lein/self-installs yourself.

  5. From the command promp run the command lein version. Leiningen should now print the version that was currently installed.

This is it! You can now use lein from the command prompt. Just type lein tutorial to get more information on how to use leiningen.

Upgrading on Windows

Upgrading isn't (yet) as smooth as on *nix-systems, but on Windows this isn't very difficult. Just follow these steps.

.
  1. To assure you have a recent .bat script (with the most recent bugfixes), download the latest one from github
  2. Replace the old .bat script with the one you downloaded
  3. Change the version number of leiningen on line 3 of the .bat if needed. By default it is already set to the newest stable version of leiningen.
  4. Run lein self-install from the command prompt or download manually in case you didn't install an HTTP client (see installation instructions).
  5. From the command promp run the command lein version. Leiningen should now print the version that was currently installed.
