Installation on Windows

To install leiningen on Windows follow these steps:

  1. Download the leiningen script. This is a .bat file which you can call from the command prompt.

    Detailed instruction

    The default name of this script is lein. If you are working with multiple versions of leiningen you are free to rename this script for a specific version, e.g. lein2. On the third line of the script there is a version variable. This is set to the newest version by default. You can change it if you want to work with a specific version of leiningen for whatever reason.

  2. Place the script on your PATH.

    Detailed instruction

    One way to do this is to make a directory for the leiningen script to reside in, e.g. c:\Program Files\leiningen. Place the script in the directory.

    Then change the PATH environment variable by appending ;c:\Program Files\leiningen to it. More details on how to change the PATH variable per version of Windows can be found here.
  3. Download wget or curl and make it available on the PATH.

    Detailed instruction

    For leiningen to work properly it needs a program that can download leiningen itself and of course dependencies of your projects. Leiningen uses either wget or curl to do this. You can make a separate directory for this software and place it on the PATH like we did with the script in step 2, but since wget.exe is a single standalone file, I tend to just place it in C:\Program Files\leiningen, since that directory is on the PATH already.

  4. Open a command prompt and run lein self-install.

    Detailed instructions

    Open a command prompt by choosing from the Start menu > Run and then enter cmd.exe. A command prompt will now show up. Enter the command lein self-install. Leiningen will now download itself into a directory %USERPROFILE%/.lein/self-installs. %USERPROFILE% is an environment variable which usually is set to your home directory. You can open this directory in the file explorer to see if leiningen succesfully downloaded itself.

This is it! You can now use lein from the command prompt. Just type lein tutorial to get more information on how to use leiningen.
